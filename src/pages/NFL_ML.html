NFL Next Gen Stats ML Model:
Data Analysis with Machine Learning

Overview

Data interpretation is at the heart of the NFL, from league records to player injury recoveries. Additionally, it powers sports betting. I'm curious how software and machine learning can take the mass amounts of data recorded during the NFL season and manipulate it to provide insights on both realms. I don't want to make money off of the software, but I want to see how numbers and effective analysis can predict outcomes.

This project focuses on scraping data from NFL statistics, including game and player data, to build a machine learning model for predicting player and team performace. The project involves data collection and parsing through scraping classes, date parsing functions and encapsulation classes to store information into a structured database. The machine learning model is a classifier trained using gini and entropy training mechanisms modeled in a decision tree structure. The project is written in Python, utilizing libraries such as requests for retrieving the JSON object, NumPy and Matplotlib for data analysis, and Pandas and Sklearn for data manipulation and model training. The database is managed using SQLite for efficient data storage and retrieval.

Currently, in the project's beginning stages, the program only outputs descriptive analytics, but once the existing raw data is parsed and organized into each of the respective tables, I can train and enhance my ML model.

Data Vis Code

My QB Passing Zone Success Rating Plot takes in a player and a game object and mimics the passing zone layout on a field and translates the raw data into meaningful performance information. The plot visualizes the QB's relative performance against the game's average across the field, highlighting strengths and potential execution deficits. The thresholds, below, average, and above, are defined by the NFL and relative to the overall game QB rating average. My plot not only shows the success level, but also the QB's individual ratings, which enhances performance evaluation capabilities by enabling players to compare their raw scores across games.
